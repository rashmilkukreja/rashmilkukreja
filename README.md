## Featured Work

### CI/CD Pipeline with Docker, ECR & EKS
Built an end-to-end deployment pipeline that builds Docker images, pushes them to AWS ECR, and deploys workloads to Amazon EKS with rolling updates and zero-downtime delivery.

### Automated EKS Deployment using Terraform
Created a production-ready EKS setup using Terraform, including VPC, IAM, OIDC, node groups, ALB controller configuration, and Kubernetes application deployment.

### Kubernetes Observability & Alerting
Implemented alerting and dashboards for Kubernetes services using Prometheus, Grafana, Alertmanager, and stakeholder notifications for faster incident response.

### Hosted Portfolio on AWS
Built and hosted a static CV/portfolio website with AWS S3, Route 53, Bootstrap, and form-based contact notifications.
